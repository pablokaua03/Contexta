# Changelog

## [1.2.0] — 2025-01-01

### Changed
- Complete UI redesign: fixed dark Entry fields, compact layout, consistent color palette
- Window icon (`icon.ico`) now loads automatically
- Fixed-width window with proper proportions — no scrolling needed
- Buttons with hover effects and disabled states
- Log area properly sized
- Progress bar thinner and less intrusive

---

## [1.1.0] — 2025-01-01

### Added
- Native `.gitignore` support
- Copy to clipboard (GUI + `--copy` CLI flag)
- Token estimation with model hints
- Git diff mode (`--diff` / `--staged`)
- AI instruction injection (GUI field + `-p` CLI flag)
- Build scripts (`build.bat`, `build.sh`)

---

## [1.0.0] — 2025-01-01

### Added
- GUI interface with dark theme (tkinter, zero dependencies)
- CLI mode
- Recursive project scanner with smart filtering
- ASCII directory tree
- Syntax-highlighted code blocks for 50+ extensions
- Multi-encoding file reader
- Hard safety limits
- Cross-platform Desktop detection
